# Estoque

### Controle de estoque

## Como rodar o projeto?

* Clone o repositório
* Execute o poetry install
* Execute o poetry shell
* Rode as migrações

```
git clone https://github.com/JohnElvisMedeirosCosta/estoque.git
cd estoque
poetry install
poetry shell
python manage.py migrate
```
